# Cover Maker

This project facilitates the creation of cover images. It was created to facilitate the generation of custom cover pictures with the following characteristics:
  - 1600x900 px.
  - A title banner of the user-chosen color, with text written by the user.
  - The banner can be located either at the top or the bottom third of the picture.

Just upload the image, select the area to use, the color of the banner, and the text for the title.
